현재의 이름인 CSS Selector은 가칭이며, 차후 변경될 수 있습니다.

## 어떤 프로젝트인가요?

CSS에서 헷갈리는 selector의 개념을 게임으로 학습하는 프로젝트입니다.  
[CSS diner](https://flukeout.github.io/)가 모티브이며, 우선 한글만 제공할 예정입니다.
